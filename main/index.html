Universal Machine

I built a Universal Machine that was an mulator for a simple computer at the machine architecture/ISA level. This was a turing-complete machine in 14 instructions written in C in a macro assembler to create executable binaries for the UM emulator. I learned to distinguish binary machine code from assembly code, how a sensible binary instruction set works and write assembly code.

Carbon FoodPrint

Carbon FoodPrint is an unfinishe project developed during a hackathon. It is designed to educate users on carbon emissions of the food industry and for users to keep track of how much carbon emissions they contribute to. Users input their daily food consumptions and the data is stored in the cloud database MongoDB. The data is put through an algorithm that calculates the total carbon emission. Bootstrap and Heroku are both used to build this web app. Facebook API works in a different version of this web app.

Image Compressor

Lossy Compression and Decompression of Ppm Image program. This project contains different commands that maps the image 0, 90, 180, 270 degrees and storing the image. I gained a deep understanding of capabilities and limitations of machine arithmetic and the binary representation of images. Data structures include unboxed representation of dynamic arrays and boxed representation of arrays, which help support the polymorphism of the image rotation program.
